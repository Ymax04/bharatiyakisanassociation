# Bharatiya Kisan Association - Web Platform

The official digital platform for the **Bharatiya Kisan Association**, designed to manage memberships, showcase field activities, and facilitate community support.

## 🚀 Features

* **Membership Management:** Integrated digital membership form with automated receipt generation.
* **Activity Gallery:** Dynamic showcase of field events and association milestones.
* **Donation System:** Secure UPI-integrated donation module for community funding.
* **Fully Responsive:** Optimized for mobile, tablet, and desktop views to reach farmers everywhere.

## 🛠️ Tech Stack

* **Frontend:** React.js with TypeScript
* **Styling:** Tailwind CSS & Shadcn/UI
* **Build Tool:** Vite
* **Deployment:** Vercel

## 📁 Project Structure

* `/src/assets` - Project images and brand assets.
* `/src/components` - Reusable UI components (Form, Gallery, Hero).
* `/src/pages` - Main page layouts.
* `/public` - Static files and favicon.
